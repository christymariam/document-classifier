# Document Classifier

A simple machine learning project that classifies text documents into categories.

## Technologies Used

- Python
- Scikit-Learn
- Streamlit

## Model

The project uses:

- TF-IDF Vectorization
- Multinomial Naive Bayes Classifier

## Example Categories

- Sports
- Technology

## How It Works

1. User enters text.
2. Text is converted into numerical features.
3. Model predicts the category.
4. Result is displayed on the webpage.
